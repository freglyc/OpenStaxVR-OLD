# OpenStaxVR

[OpenStax](https://openstax.org/) virtual reality book reader. Created in Unity 5. 

### Prerequisites

* [Unity](https://unity3d.com/) - game engine 
* [Xcode](https://developer.apple.com/xcode/) - for iOS devices

### Installing

* Download repository 
* Open in Unity
